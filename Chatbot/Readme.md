Hola es mi proyecto integrador.
La creacion de un chatbot para el manual de buenas practicas de computo en la nube
Por lo que se uso un modelo de lenguaje ya entrenado como lo es el de OpenAI, junto con una base 
de datos de vectores alojada en la nube la cual es pinecone
Dado la naturaleza de estos necesito unas API KEYS las cuales deberan cargar para que el proyecto se encuentre funcional

# Para su funcionamiento

# Se es necesario se obtenga una api de openAI
# Se cree una cuenta en la pagina de pinecone https://www.pinecone.io/
# Una vez obtenidas debe remplazarlas por la parte del codigo correspondiente
# luego instalar las dependecias necesarias, las cuales podran encontrar en "requirements.txt" 
# con el comado pip install requirements.txt dentro de una terminal
